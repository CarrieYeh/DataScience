Swiss Banknote Dataset¶

Six variables measured on 100 genuine and 100 counterfeit old Swiss 1000-franc bank notes. The data stem from Flury and Riedwyl (1988). The columns correspond to the following one label and six variables:

  1. Conterfeit - 0: genuine bank notes、1: genuine bank notes (Labels)
  1. Length - Length of the bank note
  1. Left - Height of the bank note, measured on the left
  1. Right - Height of the bank note, measured on the right
  1. Bottom - Distance of inner frame to the lower border
  1. Top - Distance of inner frame to the upper border
  1. Diagonal - Length of the diagonal

More details: Kaggle | Swiss banknote conterfeit detection

def nearest_partition(dist_sq, K):
    
    # Use [Numpy function] to return the "K Nearest Neighbor"
    # Put your code here
    enum_dist_sq = []
    ind = []
    k_nearest_neighbor = np.array([])
    
    for i in range(len(dist_sq)):
        enum_dist_sq = enumerate(dist_sq[i])
        sorted_dist_sq = sorted(enum_dist_sq,key=lambda x:x[1])[1:K+1]
        ind_list=[tup[0] for tup in sorted_dist_sq]
        ind.append(ind_list)
        for j in range(len(ind[i])):
            k_nearest_neighbor = np.concatenate([k_nearest_neighbor,data_xy[ind[i][j]]])
    k_nearest_neighbor =  k_nearest_neighbor.reshape((-1,3,2))
                
    return k_nearest_neighbor

def nearest_partition(dist_sq, K):
    
    # Use [Numpy function] to return the "K Nearest Neighbor"
    # Put your code here
    k_nearest_partition = np.array([])
    k_nearest_neighbor = np.array([])
    for i in range(len(dist_sq)):
        x = np.argpartition(dist_sq[i].ravel(),-K)[-K:]
        k_nearest_partition = np.concatenate(k_nearest_partition,x)
        n = np.unravel_index(k_nearest_partition, dist_sq.shape)
        k_nearest_neighbor = np.concatenate(k_nearest_neighbor,data_xy[n])
    
    return k_nearest_neighbor
